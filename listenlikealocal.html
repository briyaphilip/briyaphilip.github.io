Listen Like a Local

Overview

Listen Like a Local is an app that uses the Spotify API and the MediaWiki API to allow users to discover music and learn about the top artists for any country in the world. I developed this application during my Facebook University internship.


Goals:

-Create an app that allows user to discover music from all around the world

-Increase users knowledge of international artists


Team: 

Me


Scope:

2 months


Tools: 

Java, Jira


User Stories:

-User can explore Spotify playlists after entering in the country code

-User can explore artist info based on selected song


Screen Archetypes:

-Login

• Connect to Spotify account


-Stream

• View of locations user has visited/ generated playlists for

• User can click on location to reveal playlist


-Location Input

• Enter country code of country that you are interested in, currently in, or will be traveling to


-Location Overview

• Overview of what genres/songs/artists are most common in location (map view)

• Info about artist from MediaWiki API


-Playlist View

• User can now see playlists and can listen to the playlist
